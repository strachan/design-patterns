# Factory Pattern

## Quando usar

Será usado quando um método retorna uma de várias classes possíveis que compartilham uma mesma super classe.
A classe é escolhida em tempo de execução. 
Permite criar objetos sem especificar exatamente qual classe se deseja instanciar.

## Vantagens

* Não é necessário saber previamente qual classe instanciar
* Centralizar código de seleção
* O usuário não precisa saber todas as subclasses
* Encapsulamento da criação dos objetos
